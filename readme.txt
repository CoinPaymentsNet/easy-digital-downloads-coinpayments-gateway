== Description ==

Add support for CoinPayments to Easy Digital Downloads. This plugin is almost
entirely based on code provided by <a
href="https://www.coinpayments.net/index.php?cmd=merchant_tools&sub=plugins">CoinPayments</a>.

== Installation ==

1. Upload the `easy-digital-downloads-coinpayments-gateway` directory to  `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
